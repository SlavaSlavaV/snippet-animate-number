# Animate Number — анимация числовых счётчиков

WordPress PHP-сниппет для плавной анимации чисел на страницах, собранных в **Elementor Free**.
Работает на чистом Vanilla JS без jQuery.

---

## Как подключить

### Вариант 1 — через плагин Code Snippets (рекомендуется)

1. Установите плагин [Code Snippets](https://wordpress.org/plugins/code-snippets/)
2. Перейдите: **Сниппеты → Добавить новый**
3. Вставьте содержимое файла `snippet.php`
4. Установите тип: **PHP сниппет**
5. Нажмите **Сохранить и активировать**

### Вариант 2 — через WPCode

1. Установите плагин [WPCode](https://wordpress.org/plugins/insert-headers-and-footers/)
2. Перейдите: **Code Snippets → + Add Snippet**
3. Выберите тип **PHP Snippet**
4. Вставьте содержимое `snippet.php` и активируйте

### Вариант 3 — напрямую в functions.php

> [!WARNING]
> Редактирование `functions.php` напрямую может сломать сайт при синтаксической ошибке.
> Используйте этот способ только если вы уверены в своих действиях и есть резервная копия.

1. Перейдите: **Внешний вид → Редактор файлов тем** (или откройте файл через FTP/хостинг-панель)
2. Откройте `functions.php` дочерней темы
3. Вставьте содержимое `snippet.php` в конец файла

## Как использовать в Elementor

Добавьте класс `animate-num` к любому виджету через раздел **Дополнительно → CSS-класс**.

### Поддерживаемые элементы

Класс можно применить к любому HTML-тегу: `<span>`, `<p>`, `<div>`, `<h2>` и т.д.

```html
<span class="animate-num">23</span>
<span class="animate-num">100%</span>
<p class="animate-num">12+</p>
<h2 class="animate-num">1 500 000</h2>
```

> [!NOTE]
> Суффиксы и символы после числа сохраняются автоматически.
> Например, `100%` → анимирует `0%` → `100%`, суффикс `%` не теряется.

## Data-атрибуты (настройка на уровне элемента)

| Атрибут | По умолчанию | Описание |
|---|---|---|
| `data-duration` | `1500` | Длительность анимации в миллисекундах |
| `data-start` | `0` | Начальное значение счётчика |

### Примеры

```html
<!-- Анимация за 3 секунды -->
<span class="animate-num" data-duration="3000">500</span>

<!-- Старт не с нуля, а с 50 -->
<span class="animate-num" data-start="50">200</span>

<!-- Оба атрибута вместе -->
<span class="animate-num" data-duration="2000" data-start="100">999+</span>
```

---

## Как добавить data-атрибуты в Elementor

1. Выберите виджет → вкладка **Дополнительно**
2. Раздел **Атрибуты** (Elementor Pro) **или**
3. Раздел **Пользовательские атрибуты** → введите, например:
   `data-duration|2000`

> [!TIP]
> Если Elementor Free не поддерживает атрибуты в UI,
> используйте виджет **HTML** и пишите тег вручную.

## Поведение анимации

- **Триггер:** элемент попадает в видимую область экрана (20% видимости)
- **Эффект:** `easeOutQuart` — быстрый старт с плавным торможением
- **Длительность:** ~1500 мс по умолчанию
- **Повтор:** анимация запускается **только один раз** за сессию страницы
- **DOMContentLoaded:** элементы, уже видимые при загрузке, анимируются сразу
